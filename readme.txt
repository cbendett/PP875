this repository contains the scripts used to replicate the analysis of "Pervasive hybridizations in the history of wheat relatives" by Glemin et al.

all scripts can be found in the code directory and are numbered in order. most scripts were run through the CHTC's HPC system (which uses slurm) and have been formatted as such. 

there's something funky with the gitignore that let some of the result files trickle through... i didn't have time to figure out what happened there!
